## Top 5 New Best Skills in the Past Month

### 1. Veryfi

- Launch date: March 2
- Focus: invoice and receipt OCR
- Why it matters: extracts structured fields such as amount, tax ID, date, and merchant from photos, PDFs, and WhatsApp content, which makes it useful for bookkeeping and expense workflows.
- Install command: `clawhub install veryfi`

### 2. Larry

- Launch window: roughly March 8 to March 10
- Focus: AI TikTok Growth Engine
- Why it matters: covers scripts, subtitles, video generation, uploads, and growth strategy for creators and short-form content teams.
- Install command: `clawhub install larry`
- Alternate path: `clawhub.ai/OllieWazza/larry`

### 3. Skill Vetter

- Launch window: February 25 to early March
- Focus: security auditing
- Why it matters: reviews Skill code and permission patterns, flags suspicious URLs, `eval`, `exec`, file access, and other risky behavior, then returns a severity report.
- Install command: `clawhub install skill-vetter`
- Common slug: `spclaudehome/skill-vetter`

### 4. Nut Studio macOS Series

- Launch date: March 14
- Focus: macOS-local optimization
- Why it matters: optimized for faster and lower-power local execution, which fits always-on workflows on macOS.
- Install method: search `nut-studio` in ClawHub or install the matching `nut-macos-*` package directly

### 5. Felo Search / SuperAgent Series

- Attention spike: major blog coverage on March 17
- Focus: live search and content generation
- Why it matters: combines stronger real-time search with one-click PPT generation, which is useful when knowledge freshness matters.
- Install command: `clawhub install felo-search`
