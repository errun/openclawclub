This guide targets Openclaw (Moltbot / Clawdbot) safety and governance.

## Common symptoms

- Startup errors
- Behavior diverges from documentation

## Recommended fixes

1. Pin dependency versions
2. Record reproducible environment configs
3. Update regularly and run regression tests
