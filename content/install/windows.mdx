This guide covers the Openclaw (Moltbot / Clawdbot) install flow on Windows.

## Before you start

- Use PowerShell 7 or later
- Confirm proxy and corporate firewall settings

## Core steps

1. Download and extract the installer
2. Run the install script and verify the version
3. Complete the first launch and validate configuration

## Common issues

- Install fails or command not found: jump to the errors section for fixes.
