This guide covers the Openclaw (Moltbot / Clawdbot) install flow on macOS.

## Before you start

- Update to the latest system patches
- Have Terminal and administrator permissions ready

## Core steps

1. Download the installer and verify the signature
2. Run the install script
3. Configure environment variables and run startup checks

## Common issues

If you see permission prompts or commands hang, check the errors section.
