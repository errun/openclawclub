This guide targets Openclaw (Moltbot / Clawdbot) install troubleshooting.

## Possible causes

- System language is not set to Chinese
- Environment variables were not refreshed

## Fixes

1. Set system language and region
2. Restart the application
3. Confirm the language pack is installed
