This guide targets Openclaw (Moltbot / Clawdbot) install troubleshooting.

## Possible causes

- Environment variables not loaded
- Install directory not in PATH

## Fixes

1. Reopen the terminal or reboot
2. Confirm PATH includes the install directory
3. Re-run the install script
