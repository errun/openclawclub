이 가이드는 Openclaw (Moltbot / Clawdbot)의 macOS 설치 과정을 다룹니다.

## 사전 준비

- 최신 시스템 패치로 업데이트
- 터미널과 관리자 권한 준비

## 핵심 단계

1. 설치 파일을 내려받고 서명을 확인
2. 설치 스크립트를 실행
3. 환경 변수를 설정하고 시작 점검 수행

## 자주 발생하는 문제

권한 팝업이나 명령이 멈추는 경우 오류 안내 페이지를 확인하세요.
