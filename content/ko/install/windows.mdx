이 가이드는 Openclaw (Moltbot / Clawdbot)의 Windows 설치 과정을 다룹니다.

## 사전 준비

- PowerShell 7 이상 사용
- 프록시 및 사내 방화벽 설정 확인

## 핵심 단계

1. 설치 파일을 다운로드하고 압축 해제
2. 설치 스크립트를 실행하고 버전 확인
3. 첫 실행을 완료하고 설정을 검증

## 자주 발생하는 문제

- 설치 실패 또는 명령어를 찾을 수 없는 경우, 오류 안내 페이지를 확인하세요.
