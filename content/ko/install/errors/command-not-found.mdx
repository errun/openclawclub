이 가이드는 Openclaw (Moltbot / Clawdbot) 설치 문제 해결을 위한 내용입니다.

## 가능한 원인

- 환경 변수가 반영되지 않음
- 설치 경로가 PATH에 포함되지 않음

## 해결 방법

1. 터미널을 다시 열거나 재부팅
2. PATH에 설치 경로가 포함되어 있는지 확인
3. 설치 스크립트를 다시 실행
