이 가이드는 Openclaw (Moltbot / Clawdbot) 설치 문제 해결을 위한 내용입니다.

## 가능한 원인

- 시스템 언어가 한국어로 설정되지 않음
- 환경 변수가 갱신되지 않음

## 해결 방법

1. 시스템 언어와 지역 설정 확인
2. 앱 재시작
3. 언어 팩 설치 여부 확인
