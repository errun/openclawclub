## 고위험 상황

- 로그에 비밀 값을 출력
- 공개 저장소에 설정 업로드

## 권장 사항

1. 환경 변수로 비밀 값을 관리
2. CI에서 시크릿을 주입
3. 주기적으로 회전하고 권한을 감사
