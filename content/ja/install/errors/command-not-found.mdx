本ガイドは Openclaw（Moltbot / Clawdbot）のインストールトラブルシュート向けです。

## 原因

- 環境変数が読み込まれていない
- インストール先が PATH に含まれていない

## 対処

1. ターミナルを再起動するか再ログイン
2. PATH にインストール先が含まれているか確認
3. インストールスクリプトを再実行
