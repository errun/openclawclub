本ガイドは Openclaw（Moltbot / Clawdbot）の Windows インストールを対象とします。

## 事前準備

- PowerShell 7 以上を使用
- プロキシや社内ファイアウォール設定を確認

## 手順

1. インストーラーをダウンロードして展開
2. インストールスクリプトを実行しバージョンを確認
3. 初回起動を完了し設定を検証

## よくある問題

- インストール失敗やコマンドが見つからない場合は、エラー一覧を参照してください。
