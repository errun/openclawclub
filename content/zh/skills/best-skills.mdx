<Tab value="new" label="最近一个月 Top 5 新 Best Skills">

### 1. Veryfi

- 上线时间：3 月 2 日
- 定位：发票 / 收据 OCR
- 为什么值得关注：上传照片、PDF、WhatsApp 内容后，可以快速提取金额、税号、日期、商户等结构化信息，适合记账、报销和财务整理场景。
- 安装命令：`clawhub install veryfi`

### 2. Larry

- 上线时间：约 3 月 8 日到 3 月 10 日
- 定位：AI TikTok Growth Engine
- 为什么值得关注：覆盖脚本、字幕、视频生成、上传和增长策略，适合短视频创作者和内容团队。
- 安装命令：`clawhub install larry`
- 备用地址：`clawhub.ai/OllieWazza/larry`

### 3. Skill Vetter

- 上线时间：2 月 25 日到 3 月初
- 定位：安全审查员
- 为什么值得关注：主打技能代码和权限审查，会扫描未知 URL、`eval`、`exec`、文件读取等风险点，并输出分级报告。
- 安装命令：`clawhub install skill-vetter`
- 常用 slug：`spclaudehome/skill-vetter`

### 4. Nut Studio macOS 系列

- 上线时间：3 月 14 日
- 定位：macOS 本地优化技能集合
- 为什么值得关注：强调本地运行更快、更省电，适合需要长时间运行工作流的 macOS 用户。
- 安装方式：在 ClawHub 搜索 `nut-studio`，或直接安装对应的 `nut-macos-*` 技能

### 5. Felo Search / SuperAgent 系列

- 关注时间：3 月 17 日相关博客集中推荐
- 定位：实时搜索与内容生成
- 为什么值得关注：主打更强的实时搜索能力，同时支持一键生成 PPT，适合补齐知识时效性场景。
- 安装命令：`clawhub install felo-search`

<Tab value="overall" label="总榜推荐">

### 1. skill-vetter

- 关键词：安全审查 / 风险控制
- 推荐理由：适合作为安装第三方 Skills 之前的第一道检查
- 安装命令：`clawhub install skill-vetter`

### 2. self-improving-agent / capability-evolver

- 关键词：自我进化 / 偏好学习
- 推荐理由：通过记录错误、学习偏好、跨会话反思来持续优化 AI 的行为表现
- 安装命令：`clawhub install self-improving-agent`
- 备用名称：`capability-evolver`

### 3. gog

- 关键词：Google Workspace 集成
- 推荐理由：打通 Gmail、Calendar、Drive、Docs、Sheets，适合办公自动化
- 安装命令：`clawhub install gog`

### 4. agent-browser

- 关键词：浏览器自动化 / Playwright
- 推荐理由：让 AI 可以直接操作网页、填表、截图和抓取数据，是很多自动化场景的基础能力
- 安装命令：`clawhub install agent-browser`

### 5. summarize

- 关键词：网页 / PDF / YouTube 摘要
- 推荐理由：适合高频信息消费场景，能明显压缩阅读和整理时间
- 安装命令：`clawhub install summarize`

## 进阶补充

- `github`：适合开发者，用于 PR、代码审查、issue 和 repo 管理
- `tavily-search` / `Exa` / `Brave Search`：适合更强的实时联网搜索
- `notion` / `obsidian`：适合接入个人知识库，构建 RAG 记忆场景
